# Loja-Online-RCM-Lords
Este projeto simula uma loja online funcional, permitindo ao utilizador explorar um catálogo de produtos (CDs, Vinil e Merchandise), aplicar filtros, visualizar detalhes dos produtos, calcular valores totais e adicionar artigos a um carrinho simples.  O catálogo é carregado dinamicamente a partir de um ficheiro JSON, utilizando JavaScript e AJAX.
